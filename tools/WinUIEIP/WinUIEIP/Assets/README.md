# Assets Directory

This directory should contain the following image assets for the application:

- `SplashScreen.scale-200.png` - Splash screen image (620x300px)
- `LockScreenLogo.scale-200.png` - Lock screen logo (48x48px)
- `Square150x150Logo.scale-200.png` - Square logo (300x300px)
- `Square44x44Logo.scale-200.png` - Small square logo (88x88px)
- `Square44x44Logo.targetsize-24_altform-unplated.png` - Small logo variant (24x24px)
- `StoreLogo.png` - Store logo (50x50px)
- `Wide310x150Logo.scale-200.png` - Wide logo (620x300px)

These can be created using any image editor or generated using Visual Studio's asset generator.
For now, you can use placeholder images or create simple colored squares as placeholders.
